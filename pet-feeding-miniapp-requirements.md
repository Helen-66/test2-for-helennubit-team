# 宠物上门喂养小程序 - 需求文档

## 一、项目概述

### 1.1 项目背景
为宠物主人提供便捷的上门喂养服务，连接宠物主人与专业宠托师，实现基于地理位置的就近匹配、在线预约、订单管理及自动结算。

### 1.2 目标用户
- 普通用户（宠物主人）：需要上门喂养服务的养宠人群
- 宠托师：提供上门喂养服务的专业人员
- 平台管理员：负责平台运营与管理

---

## 二、功能需求

### 2.1 用户端功能

#### 2.1.1 注册与登录
- 微信一键授权登录
- 手机号绑定验证
- 用户信息完善（昵称、头像、地址）

#### 2.1.2 宠物管理
- 添加/编辑/删除宠物信息（品种、体重、年龄、性格、喂养注意事项、照片）
- 支持多宠物管理

#### 2.1.3 预约服务
- 基于用户定位，展示附近3公里范围内可用宠托师列表
- 查看宠托师详情（评分、服务次数、资质认证、用户评价）
- 选择服务时间（起止日期、每日喂养次数）
- 填写服务需求（喂食、换水、铲屎、陪玩等）
- 提交预约订单并支付

#### 2.1.4 订单管理
- 查看订单状态（待接单/已接单/进行中/已完成/已取消）
- 订单详情查看
- 订单取消（宠托师接单前可免费取消）
- 服务完成后评价与评分

#### 2.1.5 消息通知
- 订单状态变更通知
- 宠托师签到/完成通知
- 系统公告

---

### 2.2 宠托师端功能

#### 2.2.1 注册与认证
- 微信授权登录 + 手机号绑定
- 实名认证（身份证正反面、手持身份证照）
- 资质上传（宠物相关证书，选填）
- 服务地址设置（经纬度定位，覆盖3公里范围）
- 平台审核通过后方可接单

#### 2.2.2 接单管理
- 查看附近订单列表
- 接单 / 拒单操作
- 拒单需填写原因

#### 2.2.3 服务执行
- 到达用户住址后GPS签到（签到需在用户地址500米范围内）
- 上传服务过程照片/视频
- 完成服务，提交完成确认

#### 2.2.4 收入管理
- 查看收入明细
- 订单完成后T+1自动打款到微信零钱/银行卡
- 提现记录查看

#### 2.2.5 个人设置
- 服务时间段设置（可接单时段）
- 服务状态开关（在线/离线）
- 服务项目与价格设置

---

### 2.3 管理后台功能

#### 2.3.1 用户管理
- 普通用户列表与详情
- 宠托师审核（通过/驳回）
- 用户封禁/解封

#### 2.3.2 订单管理
- 订单列表与搜索
- 异常订单处理
- 退款管理

#### 2.3.3 财务管理
- 打款记录
- 平台收入统计
- 对账管理

#### 2.3.4 系统配置
- 服务品类与价格区间设置
- 平台抽成比例设置
- 消息模板管理

## 四、业务流程

### 4.1 预约流程
```
用户选择宠托师 → 填写服务需求 → 提交订单 → 支付 → 等待宠托师接单
→ 宠托师接单 → 服务时间到达 → 宠托师签到 → 执行服务 → 提交完成
→ 用户确认/自动确认 → T+1打款给宠托师
```

### 4.2 订单状态机
```
待支付 → 待接单 → 已接单 → 服务中(已签到) → 已完成 → 已结算
                ↘ 已拒单 → 重新匹配/退款
待支付 → 已取消
待接单 → 已取消(用户主动取消)
```

### 4.3 结算规则
- 用户支付金额 = 服务费 + 平台服务费(可选)
- 宠托师收入 = 服务费 × (1 - 平台抽成比例)
- 打款时间 = 订单完成时间 + 24小时
- 打款方式：微信企业付款到零钱 / 银行卡转账
